
Armor Set Naming Conventions
The armor sets are prefixed as

A1_ →  Set 1

A2_ → Set 2

A3_ → Set 3

All armor pieces are grouped under empty GameObjects:

Helmet_Equipments → Helmets from A2, A3

Body_Equipments → Body armor

Hand_Equipments → Hand armor

Leg_Equipments → Leg armor
